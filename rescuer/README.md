# Tignan Rescuer
The Tignan Rescuer app enables emergency responders to process and respond to SOS incidents.

## Responder Functions
- **Incident Reception**: Instant notifications of SOS alerts.
- **Location Tracking**: Real-time GPS sharing with the admin monitoring hub.
- **Navigation**: Map-based interface for navigating to incident coordinates.
- **Status Updates**: Incident acceptance and resolution reporting.

## Tech Stack
- **Framework**: Expo / React Native.
- **Styling**: NativeWind (Tailwind CSS).
- **Navigation**: React Navigation.
- **Mapping**: Mapbox (@rnmapbox/maps).
- **Icons**: Lucide React Native.
