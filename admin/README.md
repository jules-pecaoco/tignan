# Tignan Admin
The Tignan Admin is the central monitoring hub for managing SOS incidents and responders.

## Dashboard Functions
- **Incident Monitoring**: Real-time map view of all active SOS alerts.
- **Responder Tracking**: Update on responder status and live geographic proximity.
- **System Management**: Oversight of rescuer registrations and user profiles.

## Tech Stack
- **Framework**: React 19 + TypeScript.
- **Build Tool**: Vite 8.
- **Styling**: Tailwind CSS 4.
- **Icons**: Lucide React.
- **Backend Service**: Supabase (PostgreSQL, Realtime, Auth).
