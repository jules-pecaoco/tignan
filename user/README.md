# Tignan User
The Tignan User app provides an instant lifeline for citizens in times of emergency.

## Citizen Functions
- **SOS Activation**: Single-tap emergency alerts for immediate assistance.
- **Location Sharing**: Automated sharing of GPS coordinates with rescuers and admins.
- **Real-time Status tracking**: Tracking the arrival and identity of dispatch responders.
- **Secure Profiles**: User identification for better incident context.

## Tech Stack
- **Framework**: Expo / React Native.
- **Styling**: NativeWind (Tailwind CSS 4).
- **Navigation**: React Navigation.
- **Backend Connector**: Supabase JS.
- **Icons**: Lucide React Native.
